# 🤝 贡献指南

感谢你对 NextXi 感兴趣！🎉

## 如何贡献

### 🐛 报告 Bug

1. 到 [Issues](https://gitcode.com/nomioe/NextXi/issues) 页面
2. 点击「新建 Issue」
3. 描述问题、复现步骤、预期行为

### 💡 提出新功能

1. 在 Issues 中提出你的想法
2. 说明使用场景和预期效果
3. 如果能提供伪代码就更好了！

### 🔧 提交代码

1. Fork 本项目
2. 创建功能分支：`git checkout -b feature/my-feature`
3. 提交代码：`git commit -m "feat: 添加 xxx 功能"`
4. 推送分支：`git push origin feature/my-feature`
5. 创建 Pull Request

### 📝 提交规范

| 前缀 | 说明 |
|------|------|
| feat: | 新功能 |
| fix: | 修复 Bug |
| docs: | 文档更新 |
| refactor: | 代码重构 |
| test: | 测试相关 |
| chore: | 杂项 |

### 💡 欢迎贡献的方向

- 🆕 新的加密算法（扩展 ALGO_TABLE）
- 🔐 OXR 扩展
- 🔑 公钥私钥功能
- 📖 文档和教程
- 🐛 安全漏洞发现和修复

## 行为准则

- 友善交流 🤗
- 不喜勿喷 😊
- 鼓励提问，没有愚蠢的问题
