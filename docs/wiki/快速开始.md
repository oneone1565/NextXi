# 🚀 快速开始

## 安装

```bash
# 克隆项目
git clone https://gitcode.com/nomioe/NextXi.git
cd NextXi
```

## 基础用法

### 1. 最简单的加密

```python
from nexi import nextxi

# 自动生成 4 字节随机密钥
encrypted = nextxi("你好世界！")
print(encrypted.hex())
# 输出类似：5a8f1203e4b7c901d2a3f5e6
```

### 2. 指定密钥加密

```python
from nexi import nextxi

key = b"\\x0A\\x1B\\x2C\\x3D"
msg = "Hello NextXi!"

# 加密
encrypted = nextxi(msg, key=key)
print(f"🔐 密文: {encrypted.hex()}")

# 解密
decrypted = nextxi(encrypted, key=key, decrypt=True)
print(f"🔓 明文: {decrypted.decode()}")  # Hello NextXi!
```

### 3. Nesh 增强模式（单向加密）

```python
from addon import nextxi as nextxi_full

# Nesh 模式使用 BLAKE2b 派生密钥，不可逆
encrypted = nextxi_full("机密数据", key=b"key1", use_nesh=True)
print(f"🔮 Nesh 密文: {encrypted.hex()}")
```

### 4. 查看算法轮转计划

```python
from nexi import show_algo_plan

# 查看前 12 个字节分别用什么算法
print(show_algo_plan(12))
```

输出：
```
  字节[  0]: Add  (加法)
  字节[  1]: XOR  (异或)
  字节[  2]: Rot  (位旋转)
  字节[  3]: NSwp (半字节交换+异或)
  字节[  4]: Add  (加法)
  ...
```

## ⚠️ 注意事项

- 密钥长度建议 **4 字节**（与算法轮转周期一致）
- `use_nesh=True` 模式是**单向加密**，无法解密
- 本项目适用于教学演示、轻量数据混淆，**不建议用于高安全场景**
